# House Data
The dataset contains house price data from switzerland.

# Project Intro
The project is about the influence of predictors on the house prices. Ultimately the most important predictors should be identified.

# Course
The projects is the first assessment for the module Applied Machine Learning and Predictive Modelling 1.
